列表样式

在HTML中，对于有序列表和无序列表的列表项符号，都是使用type属性来定义

在CSS中，不管是有序列表还是无序列表，我们都是使用list-style-type属性来定义列表项符号。

列表项符号

list-style-type属性是针对ol或者ul元素的，而不是li元素。

有序列表的list-style-type属性如下：

  1. decimal:阿拉伯数字，默认值
  2. lower-roman:小写罗马数字
  3. upper-roman:大写罗马数字
  4. lower-appha:小写英文字母
  5. upper-alpha:大写英文字母

无序列表的list-style-type属性如下：

  • disc:实心圆点，默认值
  • circel:空心圆点
  • square:正方形

使用list-style-type属性来去除有序列表或无序列表的列表项符号。

  • disc:实心圆点，默认值
  • circel:空心圆点
  • square:正方形

列表项图片

可以使用list-style-image属性来定义列表项图片，也就是使用图片来代替列表项符号。语法：


            list-style-image:url(图片路径);
  • HTML
  • CSS
  • JavaScript

list-style-image属性实际上就是把列表项符号改为一张图片，而引用一张图片就要给出图片的路径。在实际开发中，像这种列表项符号，一般情况下我们不会用list-style-image属性来实现，而是使用更为高级的字体图标（iconfont）技术来实现。
